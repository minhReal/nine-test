Roulette — Cài đặt Firebase
  1. Vào console.firebase.google.com
  2. Tạo Project mới (hoặc dùng project có sẵn)
  3. Authentication → Sign-in method → Anonymous → Bật
  4. Realtime Database → Create database → chọn vùng gần nhất
  5. Rules tab → dán: {"rules":{".read":true,".write":true}} → Publish
  6. Project Settings ⚙️ → Your apps → Web → Đăng ký → copy firebaseConfig
  7. Bấm tab "Dán Config" và dán vào
Dán firebaseConfig JSON của bạn
Config được lưu vào localStorage — chỉ cần cài 1 lần
